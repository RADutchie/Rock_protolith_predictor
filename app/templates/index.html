Rock Protolith Prediction App

This app tries to predict a rocks protolith from major element geochemistry using an algorithm trained on over half a million labeled global geochemical data. When you input you major element data, the model will try to predict if your sample is either igneous or sedimentary.

How to use

You can either enter a single sample via an online form or upload a csv file containing multiple samples. Data should be in weight% oxides and must contain the below elements.

Example minimum data required in input csv, including sample identifier in column 1 and header row. Currently all Fe must be provided as total FeO
